### 1. Account Setup

3. Set privacy preferences
4. Choose categories

### 2. Dashboard Overview
- Total balance

### 3. Managing Transactions

#### Adding Transactions
1. Click "Add Transaction"
2. Enter amount
3. Select category
4. Add description
5. Choose date

#### Importing Transactions
1. Go to Import page
2. Download template
3. Fill template
4. Upload file
5. Review and confirm

### 4. Budgets

#### Creating Budgets
1. Go to Budgets page
2. Click "Add Budget"
3. Select category
4. Set limit
5. Choose period

#### Monitoring Budgets
- View progress bars
- Check remaining amounts
- Get alerts for overages
- Analyze trends
